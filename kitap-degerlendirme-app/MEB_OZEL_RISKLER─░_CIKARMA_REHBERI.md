## 🚨 KRITIK RİSK DURUMLARI (Otomatik Maksimum Risk)

Bu durumlar bulundukça **ÖZENDIRME** olmaksızın da 5/5 riski tetikler.

### 1️⃣ ÖZENDIRMELER (Her Kriteride En Yüksek Risk)

**Tanım:** Çocuğun "siz de yapabilirsiniz" mesajını alması

```
ÖZENDIRMEDEN FARKI:
┌─ "Terörist örgüt vardır" → Bilgi (Bilimsel)
├─ "Terörist örgüt kötüdür" → Görüş (Tartışılabilir)
└─ "Siz de terörist olabilirsiniz" → ÖZENDIRME ❌❌❌

┌─ "Cezaevine girmek zor" → Gerçeklik
├─ "Cezaevine gitme" → Uyarı
└─ "Cezaevine gidersen böyle yaşarsın..." → ÖZENDIRME ❌❌❌
```

**Tetikleyen Söz Yapısı:**
- "Siz de..." + eylem
- "Ben de..." + duygusal özdeşleşme
- "Böylesi..." + takip etme çağrısı
- "Dene..." + "denemek istersen..."
- "İlk adım..." + açık talimat

**Çözüm:**
```
YANLIŞ:
"Silahı oyle kullan; nisan aldığında vur."

DOĞRU ALTERNATIF 1 - Uyarı:
"Silahlar çok tehlikelidir. Çocuklar asla ellememelidir."

DOĞRU ALTERNATIF 2 - Tarihî Analiz:
"Ordu silahları devlet güvenliği için kullanır.
 Çocuklar bu sorumluluğu taşıyamaz."
```

### 2️⃣ TEROR ÖRGÜTÜ PROPAGANDASI

**Yasaklı Örgütler:**
- PKK (Kürdistan İşçi Partisi)
- DHKP-C (Devrimci Halk Kurtuluş Partisi)
- PYD (Democratic Union Party - Suriye)
- YPG (People's Protection Units)
- MLKP (Marksist Leninist Komunist Parti)
- TKP/ML (Türkiye Komünist Partisi/Marksist Leninist)

**RİSK SÜREÇİ:**
```
Seviye 1: Sadece adı geçmek → Risk: 1
Seviye 2: "İyi yönleri vardır" → Risk: 3
Seviye 3: Övgü söylemi → Risk: 4
Seviye 4: Katılım çağrısı → Risk: 5 (MAKSIMUM)
Seviye 5: Katılım + Talimatlar → Risk: 5+ (İLEYE GÖT)
```

**Uygun Yazım:**
```
OK: "PKK terör örgütü olarak yasaklanmıştır.
     Eylemlerine karşı devlet tedbirler almıştır."

HATA: "PKK gerillasının cesaret ve direnci takdire şayandır."

OK: "Tarihî olarak ayaklanmalar oluşmuş, devlet baskı yapmıştır.
     Bu durum uluslararası hukuka aykırı görülmüştür."
```

### 3️⃣ SUÇA ÖZENDIRME (Talimatlandırma)

**Riski Yüksek Yapan İfadeler:**
```
YANLIŞ (Talimatlandırma):
"Hırsızlık yaparsanız:
 1. Güvenlik kamerasını hacking'leyin
 2. Kasayı açma kodu...
 3. Kaçış yolu planı..."

DOĞRU (Hukuki Sonuç):
"Hırsızlık yapmak için de terkedilmiş bina bulunabilir.
 Ancak bu suç olup, ceza kurumu sonucunu getirir."

DOĞRU (Edebi Anlatı):
"Kahraman para lazımdı. Bir an düşündü.
 Ama babası 'dürüst yoldan geçersin' derdi. Dükkanında çalışmaya başladı."
```

**Kontrol Listesi:**
- [ ] Adım adım talimat var mı?
- [ ] "Nasıl yapılacağı" açıklanmış mı?
- [ ] Başarı durumunda övgü var mı?
- [ ] Başarısızlık/yakalanma sonu var mı?

## ⚠️ YÜKSEKRİSK DURUMLARI (Risk 3-4/5)

### 1️⃣ AYRIMALIKLIK VE NEFRET SÖYLEMI

**Risk 4 Seviyesine Ulaşan İçerik:**
```
Sistemik Ayrımcılık + Nedenlendirme:
"X ırk doğal olarak Y'ye eğilimlidir çünkü..."

Defalarca Tekrarlanan Nefret:
"Onlar berbat, berbat, berbat" (üç sözcük)

Sonuç Önerisine Kadar Giden Nefret:
"Bu insanlar hepsinden ayrılmalı" / "Temizlenmeliler"
```

**Düzeltme Yöntemi:**
```
YANLIŞ:
"Arap ırkı asalak insanlardır çünkü hırsızlıkta ustadırlar."

ARAŞTIR:
- Kaynak var mı?
- Bölgesel faktörler göz önüne alındı mı?
- Pozitif karşı örnek var mı?

DÜZELT:
"Ortadoğu'da güvenlik sorunları var.
 Batılı müdahaleler bu sorunları artırmıştır.
 Bu bölgeden gelen insanlar çoğunlukla mağdurdur."
```

### 2️⃣ ŞIDDET TAVSİYESİ (Psikiyatrik Zarar)

**Risk 4 Durumları:**
```
Fiziksel Şiddet Detaylı Tasviri:
"Kurbanın kemikleri patladi, kan akildi..." (Çok detaylı, grafikli)

Psikolojik Travma Yaratıcı Tasvirler:
"Zihinlerde kuru kan, çiğ kalp görüntüleri hep kaldı..."
(Çocuk travma görebilir)

Hayvansal Şiddet + Sempati:
"Köpeği işkenceyle öldürdü, insanların gülüştüğü sırada
 hayvan acı içinde kıvranıyordu..." (Sadist tasvir)
```

**Yaş Grubu Etkisi:**
```
Aynı Metni Farklı Yaşlara Göre Risk Değerlendirmesi:

"Samurai kılıçla vurdu ve düşmani yendi."

6-8 yaş:   Risk: 5 (Çok şiddetli, uyumsuz)
8-10 yaş:  Risk: 3 (Tarihî olaysa sorun olabilir)
10-12 yaş: Risk: 1 (Eğitsel bağlamda tartışılabilir)
```

**Düzeltme Yöntemi:**
```
YANLIŞ (6-8 yaş için):
"Güvenlik görevlisi suçluyu yakalamak için
 onu yumrukladı, kan aktı, yere devrildü."

UYGUN (6-8 yaş):
"Güvenlik görevlisi suçluyu yakaladı.
 Şimdi polis onu gözaltına aldı."

UYGUN (10-12 yaş):
"Ordu savaşta düşmanı durdurdu.
 Askerlerin cesaret ve fedakârlığı takdire şayandır."
```

### 3️⃣ CİNSELLİK VE MAHREMIYET

**Otomatik Risk 5 Durumları:**
- Çocuk cinselliği herhangi bir bağlamda
- Açık cinsel eylem tasviri
- Tecavüz anlatısı (bağlamdan bağımsız)

**Sınır Durumları (Risk 2-3):**
```
OK (Eğitsel):
"Vücudumuz ergenlikte değişiyor.
 Bu doğal ve normaldir."

OK (Romantik):
"Eller birleşti, gözler buluştu.
 Kalpleri hızlı çarpıyordu." (Belirsiz, imajinasyonlu)

UYARI (Grafik ama sınırda):
"Karşındaki kişinin güzelliğini hissetti.
 Duyguları onu harekete geçirdi." (Söylemsiz, imajinasyon)

YANLIŞ (Açık):
[Açık cinsel eylem anlatısı] - ÇIKARILMALI
```

**Revizyon Kuralı:**
- 12 yaş altında: Hiçbir cinsel içerik
- 12+ yaş: İmajinasyonlu, saygılı yazım
- Her zaman: Mahremiyet korunmalı

## 🛠️ PRAKTIK DÜZELTME ÖRNEKLERİ

### Senaryo 1: Eski Yazarın Ayrımcı Cümlesi

**BULUNDU:**
> "Çingeneler doğal olarak hırsız olmaya eğilimlidirler
> çünkü kültürleri göçebe medeniyetidir."

**ANALIZ:**
- Risk: 5 (Sistemik ayrımcılık + nedenlendirme)
- Çözüm: Sil veya tamamen yeniden yaz

**DÜZELT:**
```
ÖNCESİ:
"Çingeneler doğal olarak hırsız olmaya eğilimlidirler."

SONRASI - Seçenek 1 (Kaldır):
[Bu cümle tamamen silinir]

SONRASI - Seçenek 2 (Köklü Revizyon):
"Göçebe yaşam döneminde, farklı ekonomik sistemler
 oluşmuştur. Bunlar tarihî sosyoloji açısından
 ilginç örneklerdir. Ancak hiçbir kültür doğası gereği
 daha yüksek veya daha düşük değildir."
```

---

### Senaryo 2: Genç Yazar'ın Özendirmeli Cümlesi

**BULUNDU:**
> "Kahraman ilk molotof kokteyli yapışını böyle yaptı:
> 1. Şişeyi benzinle doldurun
> 2. Bezle kapatın..."

**ANALIZ:**
- Risk: 5 (Talimatlandırma + Özendirme)
- Çözüm: Edebi anlatım yapısına dönüştür

**DÜZELT:**
```
ÖNCESİ:
"Kahraman ilk molotof kokteyli yapışını böyle yaptı:
 1. Şişeyi benzinle doldurun
 2. Bezle kapatın
 3. Atışta doğru açı..."

SONRASI:
"Kahraman eli titreğini tuttu. Eski bilgilerini
 hatırladı. Ancak bu yol çok tehlikeliydi. Yakından
 polis sirenleri geliyordu. O, dışarıya çıktı ve
 teslim oldu. 'Daha yolum var' dedi kendi kendine."

ALTERNATİF:
"Demir çubuk, benzin, bez... Elinde tuttuğu araçlar
 onu korku içine sürükledi. Anası söylemiş olurdu:
 'Böyle yollardan bak uzak duş.' O, eli titreğini tuttu."
```

### Senaryo 3: Tarihî Ama Uyarı Gereken

**BULUNDU:**
> "Birinci Dünya Savaşında askerler cephelerde binlerce ölmüştür.
> Kurşunlarla başları kopmuş, kanları topraklara karışmıştır."

**ANALIZ:**
- Risk: 2-3 (Tarihî, ama aşırı grafik)
- Çözüm: Çok fazla detayı kaldırarak tarihî bağlamı koru

**DÜZELT:**
```
ÖNCESİ:
"Kurşunlarla başları kopmuş, kanları topraklara karışmıştır."

SONRASI:
"Savaş insanlara büyük acı vermiştir.
 Binlerce asker hayatını kaybetmiştir.
 Bu yüzden barış çok önemlidir."

ALTERNATİF:
"Savaş vahşi bir olgudur. Tarihî olarak, çatışmalar
 insanlığa ders vermiştir. Modern zamanlar barışın
 ve diplomasinin önemine daha çok işaret eder."
```

## 🔍 HIZLI KONTROL LİSTESİ

Kitaptan 10 sayfa seç, bu kontrol listesini uygula:

- [ ] **ÖZENDIRME:** "Siz de...", "Sen de...", "Dene..." var mı?
  - Hayır → Devam
  - Evet → Risk 5, sil

- [ ] **TEROR:** PKK, DHKP-C, etc. övülmüş mü?
  - Hayır → Devam
  - Evet → Risk 5, sil

- [ ] **AYRIMALIKLIK:** Bir grup sistematik aşağılanmış mı?
  - Hayır → Devam
  - Evet → Risk 4, düzelt

- [ ] **ŞİDDET:** Talimatlandırılmış şiddet var mı?
  - Hayır → Devam
  - Evet → Risk 4-5, düzelt

- [ ] **CİNSELLİK:** Çocuk cinselliği veya açık içerik var mı?
  - Hayır → Devam
  - Evet → Risk 5, sil

- [ ] **REKLAM:** Marka adı satış amacıyla yazılmış mı?
  - Hayır → Devam
  - Evet → Risk 3, düzelt

- [ ] **KUFUR:** Doğrudan küfür var mı?
  - Hayır → Devam
  - Evet → Risk 5, sil
